# RealEstate Auction (Учебный проект)

## О проекте
Современный минималистичный учебный проект аукциона недвижимости на Flask.
Лоты (properties) хранятся в файле `data.json`.

## Как запустить
1. Создайте и активируйте виртуальное окружение (рекомендуется):

   Windows:
   ```
   python -m venv venv
   venv\Scripts\activate
   ```
   macOS / Linux:
   ```
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```
   python app.py
   ```
4. Откройте в браузере: http://127.0.0.1:5000
